# 背景音乐说明

## 如何添加背景音乐

1. 将您的背景音乐文件（MP3格式）重命名为 `background.mp3`
2. 将文件放置到 `public/music/` 目录中
3. 刷新页面，点击右上角的音乐播放按钮即可播放

## 音乐文件要求

- 格式：MP3（推荐）
- 文件名：background.mp3
- 位置：public/music/background.mp3
- 建议大小：不超过 5MB

## 推荐音乐类型

- 轻音乐、背景音乐
- 城市主题音乐
- 现代感、科技感的音乐
- 与数据新闻、地铁主题相关的音乐

## 注意事项

- 确保音乐文件版权合法
- 音量适中，不要过大
- 建议使用循环播放
- 音乐应与数据新闻的严肃性相匹配
